## 1. Project Description

This project presents a comprehensive microbial community analysis based on the provided species-level abundance profiles and corresponding metadata for Batch 3.

The analysis is divided into three main parts as required by the assignment:

1.  **Q1 (Alpha Diversity):** Evaluates within-sample diversity (Shannon index) across different age categories (young, middle age, senior) and between study conditions (Control vs. Diseased). Statistical significance was assessed using Mann-Whitney U and Kruskal-Wallis tests.
2.  **Q2 (Beta Diversity):** Examines between-sample community dissimilarity using the Bray-Curtis metric. This includes PCoA visualization, PERMANOVA tests (by study condition, age category, and gender), and a non-parametric multivariate test (`adonis2`) controlling for age as a covariate.
3.  **Q3 (Differential Abundance):** Identifies specific microbial species that differ significantly between Control and Diseased groups. This involves calculating p-values (Wilcoxon), q-values (FDR correction), and Log2 Fold Change (Log2FC) for all species. Results are visualized using a volcano plot and boxplots of the top 8 differentially abundant species.

All findings, figures, and interpretations are consolidated in the `CoMeg Assignment-2 Report (Group-3).pdf`.
